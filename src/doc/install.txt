
Prerequisites:

	- gcc-c++ >= 4.0 (may work with older ones)
	- libHX >= 3.0.1
	- libxml2 >= 2.6
	- openssl (libcrypto) >= 0.9.7
	- pam >= 0.99
	- pkgconfig >= 0.19

Optional:

	- cups >= 1.3, for the "lpacct" program
	- GNU make >= 3.81 (may work with >= 3.79), for documentation
	- libldap >= 2.3, for LDAP drivers
	- libmysqlclient >= 5.0, for MySQL drivers
	- wxWidgets >= 2.7 (may work with >= 2.6), for the "Steelmill" UI
