DAUtool
=======

BESCHREIBUNG
    DAUtool ist ein Programm zur Synchronisation der </etc/passwd> mit einer
    Liste von Usern. Somit entfällt das Hinzufügen und Entefernen jenes
    einzelnen Benutzers.
    Bietet auch Support für einzelne User.

SYSTEMANFORDERUNGEN
    Linux oder ähnliche Unix-Varianten, mindestens 16 MB RAM,
    Perl v5.6.0, und das Modul Getopt::Long.

URL
